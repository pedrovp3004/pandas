# Pandas para Análise de Dados

Este repositório contém meus estudos práticos com a biblioteca **Pandas**, uma das principais ferramentas utilizadas em projetos de **Ciência de Dados e Análise de Dados em Python**.

Os notebooks foram desenvolvidos ao longo de um curso de formação em Data Science e registram exercícios e exemplos aplicando conceitos fundamentais de manipulação e tratamento de dados.

---

# Objetivo

Consolidar conhecimentos em **manipulação de dados com Pandas**, explorando operações comuns utilizadas em pipelines de análise e preparação de dados.

Entre os conceitos praticados estão:

- Criação de DataFrames
- Manipulação de colunas
- Limitação de exibição de dados
- Remoção de duplicatas
- Ordenação de dados
- Mesclagem de DataFrames
- Tratamento e limpeza de dados

Essas operações são amplamente utilizadas em processos de **ETL, análise exploratória de dados (EDA) e preparação de dados para modelos de Machine Learning**.

---
